A foundational Spring Framework project demonstrating core Dependency Injection (DI) and Inversion of Control (IoC) concepts. This project focuses on modern Java-based configuration and robust bean management.

🚀 Features
Spring Core & Context: Utilizing Spring 6.1.5 for container management.

Java Configuration: Zero XML overhead using @Configuration and @ComponentScan.

Dependency Injection: Clear examples of @Autowired for automated wiring.

Fine-grained Control: Implementation of @Qualifier to resolve bean ambiguity.

Maven Managed: Clean dependency management via pom.xml.

🛠️ Tech Stack
Java: 19

Framework: Spring Framework 6.1.5

Build Tool: Maven

IDE: IntelliJ IDEA
